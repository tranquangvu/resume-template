== Resume template
Simple project for UI coding practice

* Ruby: 2.3.1
* Rails: 4.1.0

== Setup
* Note: This is only frontend project, so don't add `config/database.yml` and don't run any `rake db:*` task

- Clone project source
- Install gem:
    ```
    gem install bundler
    bundle install
    ```
- Start server: `rails s`
